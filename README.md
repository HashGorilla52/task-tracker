# Task Tracker (Console version)

Консольное приложение для управления задачами.

## Технологии
- Java 21
- Maven
- PostgreSQL
- JDBC

## Запуск
1. Создать базу данных `task_tracker`
2. Настроить `application.properties`
3. `mvn clean compile exec:java`

## Функции
- Добавление задач
- Редактирование
- Удаление
- Просмотр всех задач
